## 📱 APP 介紹

這款 APP 的核心功能是**桌面小工具（Widget）**。  
設定好你的通勤路線後，將小工具放在桌面，不用打開 APP，就能直接看到接下來的列車班次。

適合每天固定搭乘台鐵或高鐵通勤的使用者。

### ⚠️ 重要提醒
> 本工具**無法**顯示列車即時誤點或取消資訊，請以台鐵 / 高鐵官方 APP 為準。

---

## ✨ 功能特色

| 功能 | 說明 |
|------|------|
| 🚂 台鐵時刻 | 設定起訖站，顯示上班 & 下班方向班次 |
| 🚄 高鐵時刻 | 設定高鐵上下班路線，自動切換方向 |
| 📅 星期篩選 | 可設定只在指定星期顯示（例如週一到週五） |
| 🎨 外觀自訂 | 背景色、透明度、文字顏色、色條顏色、字體大小皆可調整 |
| 🔄 每日自動同步 | 每天自動更新一次通勤時刻表，省電省流量 |
| 📋 完整車次頁 | 點擊小工具可查看當日完整班次列表 |
| 🚫 無廣告 | 純公益分享，完全免費 |

## 📲 安裝方式

### 方法一：直接下載 APK（推薦）

1. 前往 [Releases 頁面](https://github.com/blues32767/tw-train-desktop-widget/releases)
2. 下載最新版本的 `app-release.apk`
3. 在 Android 手機上允許「安裝未知來源應用程式」
4. 點擊 APK 檔案完成安裝

> **如何允許安裝未知來源？**  
> 設定 → 安全性 → 安裝未知應用程式 → 選擇你的瀏覽器或檔案管理器 → 允許

### 方法二：從 Google Play 下載（規劃中）

Google Play 上架審核中，敬請期待。

## 🛠️ 使用說明

### 第一步：設定路線

1. 開啟 APP
2. 選擇交通工具（🚂 台鐵 / 🚄 高鐵）
3. 填入**上班路線**的起站、終站與時間範圍
4. 填入**下班路線**的起站、終站與時間範圍
5. 點選「套用設定並從伺服器更新車次」

### 第二步：加入桌面小工具

1. 長按手機桌面空白處
2. 選擇「小工具」
3. 找到「台鐵｜高鐵 桌面時刻小工具」
4. 拖曳放置到桌面

### 第三步：自訂外觀（選用）

在 APP 設定頁面下方的「🎨 桌面小工具外觀設定」區塊：
- 調整背景底色與透明度
- 設定文字顏色與字體大小
- 為上班、下班、高鐵模式各自設定色條顏色
- 點選「✅ 套用外觀設定（不重新下載資料）」

---

## ⚙️ 設定說明

| 設定項目 | 說明 |
|---------|------|
| 上班 / 下班路線 | 設定台鐵或高鐵的起訖站及有效時間段 |
| 顯示班次數量 | 小工具上要顯示幾班車（1～10班） |
| 每日自動同步時間 | 系統排程更新的時間（預設 05:00） |
| 小工具顯示日期 | 勾選要顯示班次的星期（預設一至五） |
| 車種篩選 | 台鐵可選擇顯示的車種（自強、莒光、區間等） |
| 背景 / 文字外觀 | 顏色、透明度、字體大小 |
